Exercise 1 - Ranking and Window Functions

Objective

Learn how to rank records using SQL Server Window Functions.

Scenario

The Online Retail Store wants to identify the most expensive products in each category. Use SQL Server ranking functions to generate different rankings and compare how ties are handled.

Topics Covered

- OVER()
- PARTITION BY
- ORDER BY
- ROW_NUMBER()
- RANK()
- DENSE_RANK()

Tasks

- Rank products within each category.
- Compare ROW_NUMBER(), RANK(), and DENSE_RANK().
- Display the top three products from every category.
