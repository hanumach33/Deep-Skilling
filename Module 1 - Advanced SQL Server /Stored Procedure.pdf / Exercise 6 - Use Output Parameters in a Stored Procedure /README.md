Exercise 6 - Use Output Parameters in a Stored Procedure

Objective

Learn how to use OUTPUT parameters in a Stored Procedure to return calculated values.

Scenario

The Employee Management System requires a stored procedure that calculates and returns the total salary of employees in a specific department using an OUTPUT parameter.

Topics Covered

- CREATE PROCEDURE
- INPUT Parameters
- OUTPUT Parameters
- SUM()
- EXEC

Tasks

- Create a stored procedure named sp_GetTotalSalary.
- Accept DepartmentID as an input parameter.
- Return the total salary using an OUTPUT parameter.
- Execute the stored procedure.
- Display the returned value.
