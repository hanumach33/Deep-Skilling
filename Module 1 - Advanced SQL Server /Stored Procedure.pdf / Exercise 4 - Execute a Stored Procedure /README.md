Exercise 4 - Execute a Stored Procedure

Objective

Learn how to execute a Stored Procedure with input parameters in SQL Server.

Scenario

The Employee Management System requires retrieving employee details for a specific department by executing an existing stored procedure.

Topics Covered

- EXEC
- Stored Procedures
- Input Parameters

Tasks

- Execute the stored procedure.
- Pass the DepartmentID as a parameter.
- Display the employee details for the specified department.
