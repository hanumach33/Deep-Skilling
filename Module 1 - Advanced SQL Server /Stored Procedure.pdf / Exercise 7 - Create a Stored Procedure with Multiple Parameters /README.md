Exercise 7 - Create a Stored Procedure with Multiple Parameters

Objective

Learn how to create a Stored Procedure with multiple parameters to update employee salary.

Scenario

The Employee Management System requires a stored procedure to update the salary of an employee using EmployeeID and the new Salary.

Topics Covered

- CREATE PROCEDURE
- UPDATE
- Parameters
- EXEC

Tasks

- Create a stored procedure named sp_UpdateEmployeeSalary.
- Accept EmployeeID and Salary as parameters.
- Update the employee salary.
- Execute the stored procedure.
- Verify the updated employee details.
