Exercise 5 - Return Data from a Stored Procedure

Objective

Learn how to create a Stored Procedure that returns data based on input parameters.

Scenario

The Employee Management System requires a stored procedure that returns the total number of employees working in a specific department.

Topics Covered

- CREATE PROCEDURE
- Parameters
- COUNT()
- SELECT
- EXEC

Tasks

- Create a stored procedure named sp_GetEmployeeCount.
- Accept DepartmentID as an input parameter.
- Count the employees in the specified department.
- Return the total employee count.
- Execute the stored procedure.
