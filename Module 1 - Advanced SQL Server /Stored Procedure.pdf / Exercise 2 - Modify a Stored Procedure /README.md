Exercise 2 - Modify a Stored Procedure

Objective

Learn how to modify an existing Stored Procedure using ALTER PROCEDURE.

Scenario

The Employee Management System requires the existing stored procedure to display employee salary along with other employee details.

Topics Covered

- ALTER PROCEDURE
- Parameters
- SELECT Statement
- EXEC

Tasks

- Modify the existing stored procedure.
- Include the Salary column in the result.
- Execute the stored procedure.
- Verify the updated output.
