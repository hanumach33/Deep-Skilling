Exercise 1 - Create a Stored Procedure

Objective

Learn how to create a Stored Procedure in SQL Server to retrieve and insert employee information.

Scenario

The Employee Management System requires a stored procedure to retrieve employees by department and another stored procedure to insert new employee records.

Topics Covered

- CREATE PROCEDURE
- Parameters
- SELECT Statement
- INSERT Statement
- EXEC

Tasks

- Create a stored procedure to retrieve employees by DepartmentID.
- Create a stored procedure named sp_InsertEmployee.
- Execute the stored procedures.
- Verify the results.
