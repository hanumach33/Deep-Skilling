Exercise 8 - Create a Stored Procedure with Conditional Logic

Objective

Learn how to use conditional logic in a Stored Procedure to update employee salaries based on department.

Scenario

The Employee Management System requires a stored procedure that gives a bonus to employees working in a specified department.

Topics Covered

- CREATE PROCEDURE
- IF...ELSE
- UPDATE
- Parameters
- EXEC

 Tasks

- Create a stored procedure named sp_GiveBonus.
- Accept DepartmentID and BonusAmount as parameters.
- Update the salaries of employees in the specified department.
- Execute the stored procedure.
- Verify the updated employee details.
