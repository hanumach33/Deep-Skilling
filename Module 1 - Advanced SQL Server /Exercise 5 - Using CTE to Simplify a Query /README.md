Exercise 5 - Using CTE to Simplify a Query

Objective

Learn how to use Common Table Expressions (CTEs) to simplify complex SQL queries.

Scenario

The Online Retail Store wants to identify customers who have placed more than three orders. Use a CTE to calculate the total number of orders placed by each customer and retrieve only those customers who satisfy the condition.

Topics Covered

- Common Table Expressions (CTEs)
- WITH Clause
- Aggregate Functions
- COUNT()
- GROUP BY
- INNER JOIN

Tasks

- Create a CTE to count customer orders.
- Join the CTE with the Customers table.
- Display customers with more than three orders.
- Sort the results by total orders.
