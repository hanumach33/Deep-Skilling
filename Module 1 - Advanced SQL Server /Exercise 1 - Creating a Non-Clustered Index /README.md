Exercise 1 - Creating a Non-Clustered Index

Objective

Learn how to create a Non-Clustered Index to improve search performance on frequently queried columns.

Scenario

The Online Retail Store frequently searches products by ProductName. Create a Non-Clustered Index and compare query execution before and after index creation.

Topics Covered

- CREATE INDEX
- NONCLUSTERED INDEX
- Query Optimization
- Indexing

Tasks

- Execute the query before creating the index.
- Create a Non-Clustered Index on ProductName.
- Execute the same query after creating the index.
- Compare the execution performance.
