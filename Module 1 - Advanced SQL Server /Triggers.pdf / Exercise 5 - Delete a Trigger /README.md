Exercise 5 - Delete a Trigger

Objective

Learn how to delete an existing Trigger in SQL Server.

Scenario

The Employee Management System no longer requires the trigger created on the Employees table. Delete the trigger and verify that it has been removed.

Topics Covered

- DROP TRIGGER
- SQL Server Triggers
- Object Management

Tasks

- Delete the existing trigger.
- Verify that the trigger has been removed.
- Execute the SQL statements successfully.
