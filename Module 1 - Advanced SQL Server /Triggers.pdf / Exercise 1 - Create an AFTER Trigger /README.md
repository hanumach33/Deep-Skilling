Exercise 1 - Create an AFTER Trigger

Objective

Learn how to create an AFTER Trigger in SQL Server to log changes made to a table.

Scenario

The Employee Management System requires maintaining a history of salary updates. Whenever an employee's salary is modified, the change should be recorded in a separate log table.

Topics Covered

- CREATE TRIGGER
- AFTER Trigger
- UPDATE
- INSERTED Table
- Trigger Logging

Tasks

- Create the EmployeeChanges table.
- Create an AFTER UPDATE trigger.
- Log salary changes.
- Test the trigger.
