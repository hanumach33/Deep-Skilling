Exercise 6 - Create a Trigger to Update a Computed Column

Objective

Learn how to create a Trigger that updates a computed column whenever the base column changes.

Scenario

The Employee Management System requires an AnnualSalary column that is automatically updated whenever an employee's Salary is modified.

Topics Covered

- ALTER TABLE
- CREATE TRIGGER
- AFTER UPDATE
- UPDATE
- INSERTED Table

Tasks

- Add an AnnualSalary column to the Employees table.
- Create an AFTER UPDATE trigger.
- Update the AnnualSalary column whenever Salary changes.
- Test the trigger.
- Verify the updated data.
