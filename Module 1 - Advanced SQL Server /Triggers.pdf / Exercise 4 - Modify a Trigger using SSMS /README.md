Exercise 4 - Modify a Trigger using SSMS

Objective

Learn how to modify an existing Trigger in SQL Server.

Scenario

The Employee Management System requires updating the existing AFTER UPDATE trigger to include additional information whenever an employee's salary is modified.

Topics Covered

- ALTER TRIGGER
- AFTER Trigger
- UPDATE
- INSERTED Table
- DELETED Table

Tasks

- Modify the existing trigger.
- Update the trigger logic.
- Save and execute the trigger.
- Verify the updated results.
