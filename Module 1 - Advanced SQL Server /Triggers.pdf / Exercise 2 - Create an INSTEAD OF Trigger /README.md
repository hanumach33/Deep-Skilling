Exercise 2 - Create an INSTEAD OF Trigger

Objective

Learn how to create an INSTEAD OF Trigger in SQL Server to prevent deletion of records.

Scenario

The Employee Management System should prevent users from deleting employee records. Whenever a DELETE operation is attempted, the trigger should stop the deletion and display an error message.

Topics Covered

- CREATE TRIGGER
- INSTEAD OF Trigger
- DELETE
- RAISERROR

Tasks

- Create an INSTEAD OF DELETE trigger.
- Prevent deletion from the Employees table.
- Display an error message.
- Test the trigger.
