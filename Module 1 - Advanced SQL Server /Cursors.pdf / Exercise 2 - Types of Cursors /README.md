Exercise 2 - Types of Cursors

Objective

Learn how to use different types of cursors in SQL Server and understand their behavior.

Scenario

The Employee Management System requires processing employee records using different cursor types. Create STATIC, DYNAMIC, FORWARD_ONLY, and KEYSET cursors and observe their behavior.

Topics Covered

- STATIC Cursor
- DYNAMIC Cursor
- FORWARD_ONLY Cursor
- KEYSET Cursor
- FETCH
- CLOSE
- DEALLOCATE

Tasks

- Create a STATIC cursor.
- Create a DYNAMIC cursor.
- Create a FORWARD_ONLY cursor.
- Create a KEYSET cursor.
- Compare the behavior of each cursor type.
