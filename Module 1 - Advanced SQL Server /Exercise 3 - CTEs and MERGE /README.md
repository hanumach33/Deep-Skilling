Exercise 3 - CTEs and MERGE

Objective

Learn how to use Common Table Expressions (CTEs), Recursive CTEs, and the MERGE statement in SQL Server.

Scenario

The Online Retail Store wants to generate a calendar for January 2025 and synchronize product information from a staging table into the main Products table.

Topics Covered

- Common Table Expressions (CTEs)
- Recursive CTE
- MERGE Statement
- INSERT
- UPDATE
- WHEN MATCHED
- WHEN NOT MATCHED

Tasks

- Generate dates from 2025-01-01 to 2025-01-31 using a Recursive CTE.
- Create a staging table containing updated product information.
- Synchronize the staging table with the Products table using MERGE.
- Update existing products.
- Insert new products if they do not exist.
