Exercise 2 - Create a Table-Valued Function

Objective

Learn how to create a Table-Valued Function in SQL Server to return employee details based on a department.

Scenario

The Employee Management System requires a table-valued function that returns all employees belonging to a specified department.

Topics Covered

- CREATE FUNCTION
- Table-Valued Function
- RETURNS TABLE
- SELECT
- WHERE

Tasks

- Create a table-valued function named fn_GetEmployeesByDepartment.
- Accept DepartmentID as an input parameter.
- Return all employees belonging to the specified department.
- Test the function by retrieving employees from the IT department.
