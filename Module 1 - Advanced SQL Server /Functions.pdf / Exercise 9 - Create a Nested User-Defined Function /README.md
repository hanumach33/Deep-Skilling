Exercise 9 - Create a Nested User-Defined Function

Objective

Learn how to create a Nested User-Defined Function in SQL Server.

Scenario

The Employee Management System requires a function that calculates the total compensation of an employee by combining the annual salary and bonus using existing user-defined functions.

Topics Covered

- CREATE FUNCTION
- Nested User-Defined Functions
- Scalar Functions
- RETURN
- Function Calls

Tasks

- Create a function named fn_CalculateTotalCompensation.
- Use fn_CalculateAnnualSalary.
- Use fn_CalculateBonus.
- Return the total compensation.
- Test the function using employee records.
