Exercise 6 - Execute a User-Defined Function

 Objective

Learn how to execute a Scalar User-Defined Function in SQL Server.

Scenario

The Employee Management System requires the annual salary of every employee to be calculated using the fn_CalculateAnnualSalary function.

Topics Covered

- Scalar Function
- Function Execution
- SELECT
- SQL Server Functions

Tasks

- Execute the fn_CalculateAnnualSalary function.
- Calculate the annual salary for each employee.
- Display the employee details with the calculated annual salary.
