Exercise 7 - Return Data from a Scalar Function

Objective

Learn how to return data from a Scalar User-Defined Function in SQL Server.

Scenario

The Employee Management System requires the annual salary of a specific employee to be calculated using the fn_CalculateAnnualSalary function.

Topics Covered

- Scalar Function
- Function Execution
- WHERE Clause
- SELECT

Tasks

- Execute the fn_CalculateAnnualSalary function.
- Calculate the annual salary for EmployeeID = 1.
- Display the employee details with the calculated annual salary.
