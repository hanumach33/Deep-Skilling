Exercise 1 - Create a Scalar Function

Objective

Learn how to create a Scalar Function in SQL Server to calculate an employee's annual salary.

Scenario

The Employee Management System requires a scalar function that calculates the annual salary based on the monthly salary.

Topics Covered

- CREATE FUNCTION
- Scalar Function
- RETURN
- SELECT

Tasks

- Create a scalar function named fn_CalculateAnnualSalary.
- Accept Salary as an input parameter.
- Return Salary multiplied by 12.
- Test the function using employee data.
