Exercise 3 - Create a User-Defined Function

Objective

Learn how to create a User-Defined Function in SQL Server to calculate the bonus of an employee.

Scenario

The Employee Management System requires a function that calculates the employee bonus as 10% of the monthly salary.

Topics Covered

- CREATE FUNCTION
- User-Defined Function
- RETURN
- Arithmetic Expressions
- SELECT

Tasks

- Create a user-defined function named fn_CalculateBonus.
- Accept Salary as an input parameter.
- Calculate the bonus as 10% of the salary.
- Test the function using employee records.
