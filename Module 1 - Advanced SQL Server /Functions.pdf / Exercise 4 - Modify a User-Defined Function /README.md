Exercise 4 - Modify a User-Defined Function

Objective

Learn how to modify an existing User-Defined Function in SQL Server.

Scenario

The Employee Management System requires the existing bonus calculation to be updated from 10% to 15% of the employee's salary.

Topics Covered

- ALTER FUNCTION
- User-Defined Function
- RETURN
- Parameters
- SELECT

Tasks

- Modify the existing function named fn_CalculateBonus.
- Update the bonus calculation to 15% of Salary.
- Test the modified function using employee records.
