Exercise 10 - Modify a Nested User-Defined Function

Objective

Learn how to modify a Nested User-Defined Function in SQL Server.

Scenario

The Employee Management System requires the total compensation function to use the updated bonus calculation and return the revised total compensation.

Topics Covered

- ALTER FUNCTION
- Nested User-Defined Functions
- Scalar Functions
- RETURN
- Function Calls

Tasks

- Modify the function named fn_CalculateTotalCompensation.
- Use the updated fn_CalculateBonus function.
- Return the revised total compensation.
- Test the modified function using employee records.
