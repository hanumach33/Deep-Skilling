Exercise 5 - Delete a User-Defined Function

Objective

Learn how to delete an existing User-Defined Function in SQL Server.

Scenario

The Employee Management System no longer requires the fn_CalculateBonus function. Delete the function from the database and verify that it has been removed.

Topics Covered

- DROP FUNCTION
- User-Defined Function
- SQL Server

Tasks

- Delete the function named fn_CalculateBonus.
- Verify that the function has been deleted.
- Execute the SQL statements successfully.
