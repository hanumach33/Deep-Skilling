Exercise 2 - Add Computed Column - Full Name

 Objective

Learn how to create a SQL Server View with a computed column.

Scenario

The Employee Management System requires a view that displays the employee's full name by combining the first name and last name.

Topics Covered

- CREATE VIEW
- Computed Columns
- String Concatenation
- INNER JOIN

Tasks

- Create a view named vw_EmployeeFullName.
- Join Employees and Departments tables.
- Create a computed column named FullName.
- Display EmployeeID, FullName, and DepartmentName.
- Retrieve data using the created view.
