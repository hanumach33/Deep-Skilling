Exercise 1 - Create a Simple View

Objective

Learn how to create a simple SQL Server View by joining multiple tables.

Scenario

The Employee Management System requires a view that displays basic employee information along with the department name.

Topics Covered

- CREATE VIEW
- INNER JOIN
- SELECT
- SQL Server Views

Tasks

- Create a view named vw_EmployeeBasicInfo.
- Join Employees and Departments tables.
- Display EmployeeID, FirstName, LastName, and DepartmentName.
- Retrieve data using the created view.
