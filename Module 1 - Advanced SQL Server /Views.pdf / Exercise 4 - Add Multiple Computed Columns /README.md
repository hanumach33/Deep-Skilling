Exercise 4 - Add Multiple Computed Columns

Objective

Learn how to create a SQL Server View with multiple computed columns.

Scenario

The Employee Management System requires a report that displays employee details, annual salary, and bonus information using computed columns.

Topics Covered

- CREATE VIEW
- Computed Columns
- INNER JOIN
- Arithmetic Expressions
- SQL Server Views

Tasks

- Create a view named vw_EmployeeReport.
- Join Employees and Departments tables.
- Display EmployeeID.
- Create a computed FullName column.
- Calculate AnnualSalary.
- Calculate Bonus as 10% of AnnualSalary.
- Retrieve data using the created view.
