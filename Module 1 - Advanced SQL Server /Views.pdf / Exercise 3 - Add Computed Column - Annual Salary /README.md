Exercise 3 - Add Computed Column - Annual Salary

Objective

Learn how to create a SQL Server View with a computed financial column.

Scenario

The Employee Management System requires a view that calculates the annual salary of every employee based on their monthly salary.

Topics Covered

- CREATE VIEW
- Computed Columns
- Arithmetic Expressions
- SQL Server Views

Tasks

- Create a view named vw_EmployeeAnnualSalary.
- Calculate the annual salary using Salary * 12.
- Display EmployeeID, FirstName, LastName, Salary, and AnnualSalary.
- Retrieve data using the created view.
