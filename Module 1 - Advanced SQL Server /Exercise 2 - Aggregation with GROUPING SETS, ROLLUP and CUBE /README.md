# Exercise 2 - Aggregation with GROUPING SETS, ROLLUP and CUBE

Objective

Learn how to generate multiple levels of summaries using SQL Server aggregation operators.

Scenario

The Online Retail Store wants to analyze product sales by Region and Category. Generate reports using GROUPING SETS, ROLLUP, and CUBE to provide detailed totals, subtotals, and grand totals.

Topics Covered

- GROUP BY
- GROUPING SETS
- ROLLUP
- CUBE
- Aggregate Functions
- JOIN

Tasks

- Join Orders, OrderDetails, Customers, and Products.
- Display total quantity sold by Region and Category.
- Generate totals using GROUPING SETS.
- Generate subtotals using ROLLUP.
- Generate all possible combinations using CUBE.
