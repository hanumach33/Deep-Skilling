Exercise 4 - PIVOT and UNPIVOT

Objective

Learn how to transform row data into columns using PIVOT and convert columns back into rows using UNPIVOT.

Scenario

The Online Retail Store wants to generate a monthly sales report for products and later convert the report back into its original row format.

Topics Covered

- PIVOT
- UNPIVOT
- Aggregate Functions
- GROUP BY

Tasks

- Aggregate sales quantity by Product and Month.
- Display monthly sales using PIVOT.
- Convert the pivoted data back to rows using UNPIVOT.
