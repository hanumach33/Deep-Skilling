# Exercise 3 - Mock Database for Unit Tests

## Objective

Learn how to mock database operations using Moq without connecting to an actual SQL Server database.

## Prerequisites

- Visual Studio 2022
- .NET SDK
- NUnit
- NUnit Test Adapter
- Moq

## Projects

- PlayersManagerLib
- PlayerManager.Tests

## Concepts Covered

- Mocking
- Database Mocking
- Dependency Injection
- NUnit
- Moq
- Unit Testing

## Outcome

Mock database operations and verify player registration without accessing a real database.
