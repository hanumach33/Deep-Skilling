# Exercise 2 - Mock File Object for Unit Tests

## Objective

Learn how to mock file system operations using Moq without accessing the actual file system.

## Prerequisites

- Visual Studio 2022
- .NET SDK
- NUnit
- NUnit Test Adapter
- Moq

## Projects

- MagicFilesLib
- DirectoryExplorer.Tests

## Concepts Covered

- Mocking
- File System Mocking
- Dependency Injection
- Unit Testing
- NUnit
- Moq

## Outcome

Mock the Directory Explorer and validate file retrieval without reading actual files.
