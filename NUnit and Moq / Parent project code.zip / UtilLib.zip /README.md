# UtilLib

## Objective

UtilLib is a .NET Class Library that provides reusable utility methods for various applications.

The library contains helper functions that can be reused by other projects.

## Project Type

Class Library (.NET Framework)

## Classes

- Utility

## Features

- Common utility methods
- Reusable helper functions
- String manipulation
- Numeric utilities
- General-purpose methods

## Purpose

This library is intended to be referenced by multiple applications to avoid duplicate code and improve maintainability.
