# LeapYearCalculatorLib

## Objective

LeapYearCalculatorLib is a .NET Class Library that determines whether a given year is a leap year.

The library validates the input year and returns an integer based on the following rules.

## Project Type

Class Library (.NET Framework)

## Class

- LeapYearCalculator

## Method

```csharp
int CheckLeapYear(int year)
```

## Return Values

| Return Value | Description |
|--------------|-------------|
| 1 | Leap Year |
| 0 | Not a Leap Year |
| -1 | Invalid Year |

## Valid Year Range

1753 - 9999 (Inclusive)

## Features

- Validates input year.
- Detects leap years.
- Detects non-leap years.
- Returns appropriate status codes.
