# Exercise 2 - Parameterized Test Cases (Multiplication and Division)

## Objective

Write parameterized NUnit test cases to verify the multiplication and division functionality of the Calculator class. Also validate exception handling for division by zero.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- CalcLibrary
- CalcLibrary.Tests

## Concepts Covered

- TestFixture
- TestCase
- Assert.AreEqual
- Assert.Fail
- Exception Handling
- Parameterized Testing
- Unit Testing

## Outcome

Validate multiplication and division operations using multiple test cases and ensure division by zero throws the expected exception.
