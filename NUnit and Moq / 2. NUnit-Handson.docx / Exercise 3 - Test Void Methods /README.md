# Exercise 3 - Test Void Methods

## Objective

Write an NUnit test case to verify the behavior of the `AllClear()` method, which resets the calculator result to zero.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- CalcLibrary
- CalcLibrary.Tests

## Concepts Covered

- TestFixture
- Test
- SetUp
- TearDown
- Assert.AreEqual
- Testing Void Methods

## Outcome

Verify that the calculator stores the result of an addition operation and that calling the `AllClear()` method resets the result to zero.
