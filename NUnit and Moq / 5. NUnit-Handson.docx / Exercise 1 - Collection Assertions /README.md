# Exercise 1 - Collection Assertions

## Objective

Write NUnit test cases to validate different collection scenarios using CollectionAssert and Assert.That.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- CollectionsLib
- CollectionsLib.Tests

## Concepts Covered

- CollectionAssert
- Assert.That
- TestFixture
- Test
- Collection Assertions
- Equality Comparison
- Unique Collections

## Outcome

Verify collection contents, uniqueness, object existence, and collection equality using NUnit assertions.
