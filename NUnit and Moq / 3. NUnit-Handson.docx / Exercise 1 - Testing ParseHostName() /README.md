# Exercise 1 - Testing ParseHostName()

## Objective

Write NUnit test cases to validate the `ParseHostName()` method of the `UrlHostNameParser` class under different execution paths.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- UtilLib
- UtilLib.Tests

## Concepts Covered

- TestFixture
- Test
- Assert.That
- Single Assertion Rule
- Unit Testing
- Naming Conventions

## Outcome

Verify that the `ParseHostName()` method correctly extracts the hostname from valid URLs and handles invalid inputs appropriately.
