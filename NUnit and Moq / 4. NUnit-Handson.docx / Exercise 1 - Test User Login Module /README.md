# Exercise 1 - Test User Login Module

## Objective

Write NUnit test cases to validate the login functionality of the `AccountsManagerLib` project for valid credentials, invalid credentials, and missing credentials.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- AccountsManagerLib
- AccountsManagerLib.Tests

## Concepts Covered

- TestFixture
- Test
- Assert.That
- Exception Testing
- Single Assertion Rule
- Unit Testing

## Outcome

Verify that the login method returns the correct success message, failure message, and throws an `ArgumentException` when required.
