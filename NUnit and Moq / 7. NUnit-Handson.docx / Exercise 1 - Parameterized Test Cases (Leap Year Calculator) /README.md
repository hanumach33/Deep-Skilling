# Exercise 1 - Parameterized Test Cases (Leap Year Calculator)

## Objective

Write parameterized NUnit test cases to validate the Leap Year Calculator functionality for valid leap years, non-leap years, and invalid year values.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter

## Projects

- LeapYearCalculatorLib
- LeapYearCalculatorLib.Tests

## Concepts Covered

- TestFixture
- TestCase
- Parameterized Testing
- Assert.That
- Single Assertion Rule
- Unit Testing

## Outcome

Verify that the Leap Year Calculator correctly identifies leap years, non-leap years, and invalid year inputs.
