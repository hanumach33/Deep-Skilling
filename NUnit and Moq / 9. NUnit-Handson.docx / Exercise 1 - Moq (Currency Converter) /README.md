# Exercise 1 - Moq (Currency Converter)

## Objective

Write NUnit test cases for the `USDToEuro` method by mocking the external exchange rate service using the Moq framework.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter
- Moq

## Projects

- ConverterLib
- ConverterLib.Tests

## Concepts Covered

- Moq
- Mock Objects
- Dependency Injection
- NUnit
- Assert.That
- Unit Testing
- TestFixture
- Test

## Outcome

Verify that the `USDToEuro` method correctly converts US Dollars to Euros by mocking the external exchange rate service.
