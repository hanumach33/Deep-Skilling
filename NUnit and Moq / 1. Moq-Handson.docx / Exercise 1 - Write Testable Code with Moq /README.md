# Exercise 1 - Write Testable Code with Moq

## Objective

Create testable code using Dependency Injection and Moq.

## Prerequisites

- Visual Studio 2022
- .NET SDK
- NUnit
- NUnit Test Adapter
- Moq

## Projects

- CustomerCommLib
- CustomerComm.Tests

## Concepts Covered

- Dependency Injection
- Constructor Injection
- Mocking
- Unit Testing
- NUnit
- Moq

## Outcome

Mock the email sender and verify business logic without sending an actual email.
