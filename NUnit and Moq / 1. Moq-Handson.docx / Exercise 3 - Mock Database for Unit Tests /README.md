# Exercise 3 - Mock Database for Unit Tests

## Objective

Learn how to mock database operations using Moq without connecting to a real database.

## Prerequisites

- Visual Studio 2022
- .NET SDK
- NUnit
- NUnit3TestAdapter
- Moq

## Projects

- PlayersManagerLib
- PlayerManager.Tests

## Concepts Covered

- Moq
- Database Mocking
- Dependency Injection
- NUnit
- Unit Testing

## Outcome

Mock database operations and verify player registration without accessing an actual SQL Server database.
