# Exercise 2 - Mock File Object for Unit Tests

## Objective

Learn how to mock file system operations using Moq without accessing the actual file system.

## Prerequisites

- Visual Studio 2022
- .NET SDK
- NUnit
- NUnit3TestAdapter
- Moq

## Projects

- MagicFilesLib
- DirectoryExplorer.Tests

## Concepts Covered

- Moq
- Mock Objects
- Dependency Injection
- NUnit
- Unit Testing

## Outcome

Mock directory operations and verify the returned files without depending on the actual file system.
