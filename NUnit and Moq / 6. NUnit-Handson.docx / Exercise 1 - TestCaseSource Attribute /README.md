# Exercise 1 - TestCaseSource Attribute

## Objective

Write NUnit test cases using the `TestCaseSource` attribute to validate the season returned for different months while minimizing duplicate test code.

## Prerequisites

- Visual Studio 2022
- .NET Framework
- NUnit
- NUnit3TestAdapter
- Moq

## Projects

- FourSeasonsLib
- FourSeasonsLib.Tests

## Concepts Covered

- TestFixture
- Test
- TestCaseSource
- Assert.That
- Parameterized Testing
- Single Assertion Rule

## Outcome

Validate the season corresponding to each month using a single parameterized test method and the `TestCaseSource` attribute.
