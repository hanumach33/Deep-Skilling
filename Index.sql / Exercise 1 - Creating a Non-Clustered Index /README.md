Exercise 1 - Creating a Non-Clustered Index

Objective

Learn how to create a Non-Clustered Index to improve query performance in SQL Server.

Scenario

The Online Retail Store frequently searches products using the ProductName column. Create a Non-Clustered Index and observe the improvement in query execution.

Topics Covered

- CREATE INDEX
- NONCLUSTERED INDEX
- Query Optimization
- SQL Server Indexes

Tasks

- Execute the query before creating the index.
- Create a Non-Clustered Index on ProductName.
- Execute the query again.
- Compare the execution performance.
