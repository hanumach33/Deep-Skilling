Exercise 3 - Creating a Composite Index

Objective

Learn how to create a Composite Index in SQL Server to improve query performance when filtering or sorting using multiple columns.

Scenario

The Online Retail Store frequently searches products by Category and Price. Create a Composite Index on these columns and compare query execution before and after index creation.

Topics Covered

- CREATE INDEX
- Composite Index
- Query Optimization
- Execution Plan

Tasks

- Execute the query before creating the index.
- Create a Composite Index on Category and Price.
- Execute the query again.
- Compare the execution performance.
