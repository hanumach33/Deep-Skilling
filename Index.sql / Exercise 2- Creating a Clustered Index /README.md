Exercise 7 - Creating a Clustered Index

Objective

Learn how to create a Clustered Index in SQL Server to organize table data physically and improve query performance.

Scenario

The Online Retail Store frequently retrieves products based on ProductID. Create a Clustered Index on the ProductID column and compare query execution before and after index creation.

Topics Covered

- CREATE CLUSTERED INDEX
- Clustered Index
- Query Optimization
- Execution Plan

Tasks

- Execute the query before creating the index.
- Create a Clustered Index on ProductID.
- Execute the query again.
- Compare the execution performance.
