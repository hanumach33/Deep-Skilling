angular.json
Contains Angular project configuration including build, serve and test settings.

tsconfig.json
Base TypeScript configuration used throughout the project.

tsconfig.app.json
TypeScript configuration specifically for the Angular application.

package.json
Contains project metadata, dependencies and npm scripts.

src/main.ts
Entry point of the Angular application that bootstraps the root component.

src/app/app.config.ts
Contains application-wide providers and routing configuration.

src/app/app.component.ts
Root component of the Angular application.

src/index.html
Main HTML page that hosts the Angular application.
