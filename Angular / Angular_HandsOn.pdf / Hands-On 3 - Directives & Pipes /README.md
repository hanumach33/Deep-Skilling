# Hands-On 3 - Directives & Pipes

## Objective

Learn Angular built-in directives, attribute directives, custom directives, and custom pipes.

---

## Topics Covered

- *ngIf
- *ngFor
- *ngSwitch
- ngClass
- ngStyle
- Custom Directive
- Custom Pipe
- HostListener
- PipeTransform

---

## Components

- CourseListComponent
- CourseCardComponent

---

## Generate Directive

```bash
ng generate directive directives/highlight
```

---

## Generate Pipe

```bash
ng generate pipe pipes/credit-label
```
