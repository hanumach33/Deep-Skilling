# Hands-On 10 - Unit Testing Angular Applications

## Objective

Learn how to write unit tests for Angular components, services, and HTTP requests using Jasmine, Karma, TestBed, and HttpTestingController.

---

## Topics Covered

- Jasmine
- Karma
- TestBed
- Component Testing
- Service Testing
- HttpTestingController
- Mocking HTTP Requests
- Code Coverage

---

## Execute Tests

```bash
ng test
```

---

## Generate Code Coverage

```bash
ng test --code-coverage
```

Coverage report will be generated inside:

```
coverage/
```
