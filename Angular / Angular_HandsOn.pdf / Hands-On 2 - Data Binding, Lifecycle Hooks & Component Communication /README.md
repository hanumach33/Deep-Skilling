# Hands-On 2 - Data Binding, Lifecycle Hooks & Component Communication

## Objective

Learn Angular data binding techniques, lifecycle hooks, and communication between parent and child components.

---

## Topics Covered

- Interpolation
- Property Binding
- Event Binding
- Two-Way Binding
- ngOnInit
- ngOnChanges
- ngOnDestroy
- @Input
- @Output
- EventEmitter

---

## Components

- HomeComponent
- CourseListComponent
- CourseCardComponent
