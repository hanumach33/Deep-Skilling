Exercise 1 - Inventory Management System

Objective

Understand the importance of Data Structures and Algorithms by developing a simple Inventory Management System.

Scenario

A warehouse needs an inventory management system to efficiently store, retrieve, update, and delete product information.

Topics Covered

- Data Structures
- Algorithms
- ArrayList Concept
- HashMap Concept
- Time Complexity
- CRUD Operations

Tasks

- Create a Product class.
- Store products in a suitable data structure.
- Add new products.
- Update existing products.
- Delete products.
- Display all products.
- Analyze the time complexity of each operation.
