Exercise 5 - Task Management System

Objective

Understand the implementation of a Singly Linked List by developing a Task Management System.

Scenario

A company requires a task management system where tasks can be added, searched, traversed, and deleted efficiently using a linked list.

Topics Covered

- Singly Linked List
- Dynamic Memory Allocation
- Node Traversal
- Searching
- Deletion
- Time Complexity

Tasks

- Create a Task class.
- Implement a Singly Linked List.
- Add tasks.
- Search tasks.
- Display all tasks.
- Delete tasks.
- Analyze the time complexity.
