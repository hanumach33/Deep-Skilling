Exercise 4 - Employee Management System

Objective

Understand array representation by implementing an Employee Management System using arrays.

Scenario

A company needs an Employee Management System to efficiently store employee records using arrays. The application should support adding, searching, traversing, and deleting employee records.

Topics Covered

- Arrays
- Array Representation
- Searching
- Traversing
- Deletion
- Time Complexity

Tasks

- Create an Employee class.
- Store employees in an array.
- Add employees.
- Search employees.
- Traverse all employees.
- Delete employees.
- Analyze the time complexity of each operation.
