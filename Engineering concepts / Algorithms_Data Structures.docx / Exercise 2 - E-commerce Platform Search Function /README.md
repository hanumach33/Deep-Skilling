Exercise 2 - E-commerce Platform Search Function

Objective

Understand searching algorithms and compare the performance of Linear Search and Binary Search.

Scenario

An e-commerce platform requires a fast product search system. Implement both Linear Search and Binary Search to search products by Product ID.

Topics Covered

- Big O Notation
- Linear Search
- Binary Search
- Arrays
- Sorting Algorithms

Tasks

- Create a Product class.
- Store products in an array.
- Implement Linear Search.
- Sort the array.
- Implement Binary Search.
- Compare both algorithms.
