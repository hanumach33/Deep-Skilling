Exercise 3 - Sorting Customer Orders

Objective

Understand and compare different sorting algorithms by sorting customer orders based on their total price.

Scenario

An e-commerce platform needs to sort customer orders according to their total price so that high-value orders can be processed first.

Topics Covered

- Bubble Sort
- Quick Sort
- Divide and Conquer
- Time Complexity
- Sorting Algorithms

Tasks

- Create an Order class.
- Store customer orders in an array.
- Implement Bubble Sort.
- Implement Quick Sort.
- Display the sorted orders.
- Compare the performance of both sorting algorithms.
