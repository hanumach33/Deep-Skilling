Exercise 6 - Library Management System

Objective

Understand the implementation and comparison of Linear Search and Binary Search algorithms.

Scenario

A library management system allows users to search for books by title. The application demonstrates both Linear Search and Binary Search to compare their efficiency.

Topics Covered

- Linear Search
- Binary Search
- Arrays
- Sorting
- Time Complexity

Tasks

- Create a Book class.
- Store books in an array.
- Implement Linear Search.
- Sort books by title.
- Implement Binary Search.
- Compare the performance of both search algorithms.
