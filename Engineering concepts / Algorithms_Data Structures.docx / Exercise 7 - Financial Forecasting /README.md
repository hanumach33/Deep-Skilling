Exercise 7 - Financial Forecasting

Objective

Understand recursion by implementing a financial forecasting application that predicts future values based on a fixed annual growth rate.

Scenario

A financial forecasting tool predicts future investment values using a recursive algorithm. The exercise demonstrates how recursion works and discusses optimization techniques.

Topics Covered

- Recursion
- Mathematical Modeling
- Financial Forecasting
- Time Complexity
- Optimization

Tasks

- Create a recursive function.
- Calculate the future value of an investment.
- Accept the initial amount, growth rate, and number of years.
- Display the predicted future value.
- Analyze the time complexity.
- Discuss optimization using Dynamic Programming.
