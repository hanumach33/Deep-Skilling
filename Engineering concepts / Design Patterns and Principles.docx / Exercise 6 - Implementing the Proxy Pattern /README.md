# Exercise 6 - Implementing the Proxy Pattern

## Objective

Learn how to implement the Proxy Design Pattern to provide lazy initialization and caching for remote images.

## Scenario

An image viewer application loads images from a remote server. To improve performance, images should be loaded only when required. The Proxy Pattern provides lazy loading and caches the image object.

## Topics Covered

- Proxy Pattern
- Subject Interface
- Real Subject
- Proxy
- Lazy Initialization
- Caching

## Tasks

- Create the Image interface.
- Create the RealImage class.
- Create the ProxyImage class.
- Implement lazy initialization.
- Test the Proxy Pattern implementation.
