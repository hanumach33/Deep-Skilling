# Exercise 11 - Implementing Dependency Injection

## Objective

Learn how to implement Dependency Injection (DI) using constructor injection to reduce coupling between classes.

## Scenario

A customer management application requires a service class that depends on a repository class. Instead of creating the repository inside the service, it is injected through the constructor.

## Topics Covered

- Dependency Injection
- Constructor Injection
- Interface
- Loose Coupling
- Inversion of Control

## Tasks

- Create the CustomerRepository interface.
- Implement CustomerRepositoryImpl.
- Create CustomerService.
- Inject CustomerRepository using constructor injection.
- Retrieve customer information.
- Test the implementation.
