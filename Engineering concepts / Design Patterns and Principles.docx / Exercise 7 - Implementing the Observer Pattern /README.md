# Exercise 7 - Implementing the Observer Pattern

## Objective

Learn how to implement the Observer Design Pattern to notify multiple clients whenever stock prices change.

## Scenario

A stock market monitoring application needs to notify registered clients whenever a stock price changes.

## Topics Covered

- Observer Pattern
- Subject
- Observer
- Register Observer
- Remove Observer
- Notify Observers

## Tasks

- Create the Stock interface.
- Create the Observer interface.
- Implement StockMarket.
- Implement MobileApp and WebApp observers.
- Register observers.
- Notify observers when stock prices change.
