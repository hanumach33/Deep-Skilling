# Exercise 9 - Implementing the Command Pattern

## Objective

Learn how to implement the Command Design Pattern to encapsulate requests as objects.

## Scenario

A home automation system allows users to turn devices ON and OFF using a remote control. The Command Pattern decouples the requester from the receiver.

## Topics Covered

- Command Pattern
- Command Interface
- Concrete Commands
- Invoker
- Receiver

## Tasks

- Create the Command interface.
- Create LightOnCommand and LightOffCommand.
- Create the Light class.
- Create the RemoteControl class.
- Execute commands using the RemoteControl.
- Test the implementation.
