# Exercise 3 - Implementing the Builder Pattern

## Objective

Learn how to implement the Builder Design Pattern to create complex objects with optional attributes.

## Scenario

A computer manufacturing company needs to build different computer configurations without creating multiple constructors. The Builder Pattern simplifies object creation.

## Topics Covered

- Builder Pattern
- Static Nested Builder Class
- Method Chaining
- Object Construction
- Encapsulation

## Tasks

- Create a Computer class.
- Create a static nested Builder class.
- Implement methods to set CPU, RAM, and Storage.
- Implement the build() method.
- Create different Computer configurations.
