Lab 6 - Updating and Deleting Records

Objective

Learn how to update and delete records using Entity Framework Core 8.0.

Scenario

The Retail Inventory System needs to update product prices and remove discontinued products from the database.

Topics Covered

- FirstOrDefaultAsync()
- Update Entity
- Remove()
- SaveChangesAsync()
- Async Programming

Tasks

- Retrieve a product by name.
- Update the product price.
- Save the changes.
- Retrieve a product to delete.
- Delete the product.
- Save the changes.
