Lab 1 - Understanding ORM with a Retail Inventory System

Objective

Understand Object-Relational Mapping (ORM) and learn how Entity Framework Core 8.0 maps C# objects to SQL Server tables.

Scenario

A retail store wants to build an inventory management system to manage products, categories, and stock levels. EF Core 8.0 will be used to simplify database operations.

Topics Covered

- Object Relational Mapping (ORM)
- Entity Framework Core 8.0
- EF Core vs EF6
- .NET Console Application
- EF Core Packages

Tasks

- Understand ORM concepts.
- Compare EF Core and EF6.
- Learn EF Core 8.0 features.
- Create a .NET Console application.
- Install EF Core packages.
