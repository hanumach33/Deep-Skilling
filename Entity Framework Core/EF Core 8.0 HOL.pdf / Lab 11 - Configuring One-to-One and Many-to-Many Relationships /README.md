Lab 11 - Configuring One-to-One and Many-to-Many Relationships

Objective

Learn how to configure One-to-One and Many-to-Many relationships using Entity Framework Core 8.0.

Scenario

The Retail Inventory System needs to store warranty information for each product and allow products to have multiple tags such as "On Sale" and "New Arrival".

Topics Covered

- One-to-One Relationship
- Many-to-Many Relationship
- Fluent API
- HasOne()
- WithOne()
- HasForeignKey()

Tasks

- Create ProductDetail entity.
- Create Tag entity.
- Configure One-to-One relationship.
- Configure Many-to-Many relationship.
- Verify the relationships.

Files

- Product.cs
- ProductDetail.cs
- Tag.cs
- AppDbContext.cs
