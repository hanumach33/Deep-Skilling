Lab 3 - Using EF Core CLI to Create and Apply Migrations

Objective

Learn how to create and apply Entity Framework Core migrations to generate the database schema.

Scenario

The Retail Inventory System needs to create the SQL Server database based on the entity models using EF Core Migrations.

Topics Covered

- EF Core CLI
- Migrations
- Database Update
- SQL Server
- Schema Generation

Tasks

- Install the EF Core CLI.
- Create the initial migration.
- Apply the migration.
- Verify the database tables.
