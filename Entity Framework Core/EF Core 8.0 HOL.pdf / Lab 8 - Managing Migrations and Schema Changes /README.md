Lab 8 - Managing Migrations and Schema Changes

Objective

Learn how to update the database schema using Entity Framework Core 8.0 Migrations.

Scenario

The Retail Inventory System needs to track product inventory. A new field named StockQuantity must be added to the Product model and reflected in the SQL Server database.

Topics Covered

- Entity Framework Core Migrations
- Model Changes
- Database Schema Updates
- EF Core CLI

Tasks

- Add StockQuantity to the Product model.
- Create a new migration.
- Apply the migration.
- Verify the updated database schema.
