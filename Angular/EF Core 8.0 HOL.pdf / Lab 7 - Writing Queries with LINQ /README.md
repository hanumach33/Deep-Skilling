Lab 7 - Writing Queries with LINQ

Objective

Learn how to write LINQ queries in Entity Framework Core 8.0 to filter, sort, and project data.

Scenario

The Retail Inventory System needs to generate reports by filtering products, sorting them by price, and displaying only the required information.

Topics Covered

- LINQ
- Where()
- OrderByDescending()
- Select()
- DTO Projection
- ToListAsync()

Tasks

- Retrieve products with Price greater than 1000.
- Sort products in descending order by Price.
- Project the result into DTO objects.
- Display the results.
