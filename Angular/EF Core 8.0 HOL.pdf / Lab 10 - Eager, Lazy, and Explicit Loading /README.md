Lab 10 - Eager, Lazy, and Explicit Loading

Objective

Learn the different loading strategies in Entity Framework Core 8.0 for retrieving related data.

Scenario

The Retail Inventory System needs to display products along with their categories. This lab demonstrates Eager Loading, Explicit Loading, and Lazy Loading.

Topics Covered

- Eager Loading
- Explicit Loading
- Lazy Loading
- Include()
- LoadAsync()
- UseLazyLoadingProxies()

Tasks

- Retrieve products with categories using Eager Loading.
- Load related category data using Explicit Loading.
- Configure Lazy Loading.
- Display the retrieved data.
