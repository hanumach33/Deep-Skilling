Lab 2 - Setting Up the Database Context for a Retail Store

Objective

Learn how to configure Entity Framework Core 8.0 DbContext and connect a .NET application to SQL Server.

Scenario

The retail store wants to store product and category information in a SQL Server database using Entity Framework Core.

Topics Covered

- Entity Framework Core 8.0
- DbContext
- DbSet
- SQL Server Connection
- Model Classes

Tasks

- Create Category and Product models.
- Create AppDbContext.
- Configure SQL Server connection.
- Create Program.cs to test the connection.

Files

- Category.cs
- Product.cs
- AppDbContext.cs
- Program.cs
