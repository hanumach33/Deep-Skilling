Lab 4 - Inserting Initial Data into the Database

Objective

Learn how to insert initial records into the database using Entity Framework Core 8.0.

Scenario

The Retail Inventory System requires initial categories and products to be inserted into the SQL Server database.

Topics Covered

- AddAsync()
- AddRangeAsync()
- SaveChangesAsync()
- Entity Relationships
- Async Programming

Tasks

- Create Category objects.
- Insert Categories.
- Create Product objects.
- Associate Products with Categories.
- Save the records into the database.
- Verify the inserted data.
