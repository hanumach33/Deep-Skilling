Lab 5 - Retrieving Data from the Database

Objective

Learn how to retrieve data from a SQL Server database using Entity Framework Core 8.0.

Scenario

The Retail Inventory System needs to display product information on the dashboard by retrieving records from the database.

Topics Covered

- FindAsync()
- FirstOrDefaultAsync()
- ToListAsync()
- LINQ
- Async Queries

Tasks

- Retrieve all products.
- Find a product by its ID.
- Retrieve the first expensive product.
- Display the results.
