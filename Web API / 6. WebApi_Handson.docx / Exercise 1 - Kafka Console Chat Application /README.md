# Exercise 1 - Kafka Console Chat Application

## Objective

Build a Console Chat Application using Apache Kafka.

One console application acts as a Producer and another acts as a Consumer to exchange chat messages.

---

## Concepts Covered

- Apache Kafka
- Producer
- Consumer
- Topics
- Brokers
- Zookeeper
- Streaming Messages
- Console Application

---

## Prerequisites

- .NET 6/7/8
- Apache Kafka
- Zookeeper
- Confluent.Kafka NuGet Package

Install Package

dotnet add package Confluent.Kafka
