# Exercise 2 - Kafka Windows Forms Chat Application

## Objective

Develop a Windows Forms Chat Application using Apache Kafka.

One Windows Forms application publishes messages to a Kafka topic while another application consumes and displays them.

---

## Concepts Covered

- Apache Kafka
- Windows Forms
- Producer
- Consumer
- Kafka Topic
- Real-Time Messaging

---

## Prerequisites

- Visual Studio
- .NET Windows Forms
- Apache Kafka
- Zookeeper
- Confluent.Kafka NuGet Package

Install Package

dotnet add package Confluent.Kafka
