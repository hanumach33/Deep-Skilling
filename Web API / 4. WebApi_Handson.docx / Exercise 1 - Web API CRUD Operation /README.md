# Exercise 1 - Web API CRUD Operation

## Objective

Implement the Update (PUT) operation in an ASP.NET Core Web API using a hardcoded list of employees.

---

## Scenario

Update employee information based on the employee ID received through the URL and the employee details received in the request body.

Validate the employee ID before updating the record.

---

## Concepts Covered

- HTTP PUT
- CRUD Operations
- FromBody Attribute
- IActionResult
- ActionResult<T>
- BadRequest
- Ok Result
- Swagger
- POSTMAN

---

## Prerequisites

- ASP.NET Core Web API
- Employee Model
- EmployeeController
