# Exercise 1 - Web API using Custom Model Class

## Objective

Create a custom Employee model and expose it through an ASP.NET Core Web API.

---

## Scenario

Create an Employee model containing Department and Skill information. Create an EmployeeController that returns a list of Employee objects and supports Read and Write operations.

---

## Concepts Covered

- Model Class
- Nested Objects
- Collection Properties
- HttpGet
- HttpPost
- HttpPut
- AllowAnonymous
- ProducesResponseType
- FromBody

---
