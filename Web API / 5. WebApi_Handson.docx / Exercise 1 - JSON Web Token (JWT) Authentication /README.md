# Exercise 1 - JSON Web Token (JWT) Authentication

## Objective

Implement JWT Authentication in ASP.NET Core Web API.

Configure JWT authentication, create an AuthController to generate JWT tokens, and secure the application using authentication middleware.

---

## Concepts Covered

- JWT Authentication
- Bearer Token
- Claims
- Symmetric Security Key
- Authentication Middleware
- AllowAnonymous
- JwtBearer
- Token Validation

---

## Prerequisites

- ASP.NET Core Web API
- Microsoft.AspNetCore.Authentication.JwtBearer package

---

- Program.cs
- Controllers/AuthController.cs
