# Exercise 2 - Test Web API using POSTMAN

## Objective

Learn how to test ASP.NET Core Web API endpoints using the POSTMAN tool.

---

## Scenario

You have already created an Employee Web API. Now test the GET endpoint using POSTMAN and verify the response body and HTTP status code.

---

## Concepts Covered

- POSTMAN
- HTTP GET
- Request URL
- Response Body
- Status Code
- JSON Response

---

## Prerequisites

- Exercise 1 completed
- Employee Controller created
- ASP.NET Core Web API running
