# Exercise 3 - Modify the Controller Route to 'Emp' and Test Using POSTMAN

## Objective

Learn how to customize the route of an ASP.NET Core Web API controller using the `Route` attribute and verify the changes using POSTMAN.

---

## Scenario

Instead of accessing the Employee controller using the default route (`api/employee`), modify the controller route to `api/emp` and test the endpoint through POSTMAN.

---

## Concepts Covered

- Route Attribute
- Custom Routing
- HTTP GET
- POSTMAN
- ASP.NET Core Web API
- Controller Routing

---

## Prerequisites

- Exercise 1 completed
- Exercise 2 completed
- EmployeeController created
